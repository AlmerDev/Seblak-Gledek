# Seblak Gledek Website
Website Seblak Gledek adalah website sederhana untuk promosi dan penjualan seblak dengan konsep topping bebas. Website ini dibuat untuk menampilkan menu, menarik pelanggan, dan mempermudah proses pemesanan.

## Fitur

* Landing page dengan branding Seblak Gledek
* Tampilan menu (beberapa varian seblak)
* Konsep pilih topping sesuai selera
* Tombol pemesanan (WhatsApp / kontak)
* Responsive (bisa dibuka di HP & desktop)

## Teknologi

* HTML
* CSS
* JavaScript
* Node.js (opsional untuk backend)

## Struktur Project

* public/
  * index.html
  * css/
  * js/
* images/
* server.js

## Cara Menjalankan

1. Clone repository
2. Install dependency:
   npm install
3. Jalankan server:
   node server.js
4. Buka di browser:
   [http://localhost:3000](http://localhost:3000)

## Deployment

Project ini bisa di-deploy menggunakan:

* Vercel (untuk frontend saja)
* Railway / Render (untuk Node.js)
* VPS (untuk full control)

## Tujuan

Project ini dibuat untuk membantu usaha kecil seperti Seblak Gledek agar bisa go online dan menjangkau lebih banyak pelanggan.
